# GT Backend Project

Projeto de Backend desenvolvido no curso Geracao Tech 3.0.

## Tecnologias

- Node.js
- Express
- PostgreSQL
- Sequelize
- JWT

## Instalação

Clone o repositório:

`git clone https://github.com/cmsisnando/gt-backend-project`

`cd gt-backend-project`

Instale as dependências:

`npm install`

## Variáveis de Ambiente

Crie um arquivo .env na raiz do projeto:

PORT=

JWT_SECRET=

JWT_EXPIRES_IN=

DB_HOST=

DB_USER=

DB_PASSWORD=

DB_NAME=

DB_PORT=

DB_DIALECT=

## Executando o Projeto

Modo desenvolvimento:

`npm run dev`

Modo produção:

`npm start`
